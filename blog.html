Important FAQ

What is the Document Object Model (DOM)?
The DOM is a programming interface that allows scripts to access and manipulate the content, structure, and style of HTML and XML documents.
How do you select an element from DOM?
1. By ID: document.getElementById("myElement")
2. By Class Name: document.getElementsByClassName ("myClass")
3. By Tag Name: document.getElementsByTagName("p")
4. By CSS Selector: document.querySelector("#myDiv .myClass")
5. By Name: document.getElementsByName("myInput")
What is event delegation in the context of the DOM, and why is it useful ?
Event delegation is a technique where you attach a single event listener to a parent element to handle events for all its current and future descendants, rather than attaching individual listeners to each child.
It's useful for:
1. Efficiency: Reduces the number of event listeners, improving performance.
2. Dynamic Elements: Works for elements added to the DOM after the initial setup.
3. Cleanliness: Keeps code organized and avoids redundant listeners.
How do you manipulate an elements attributes and styles using the DOM?
Attributes
1. Get: element.getAttribute("src")
2. Set: element.setAttribute("src", "new-image.jpg")
3. Remove: element.removeAttribute("disabled")

Styles
Directly: element.style.color = "red"
Via Class:
element.classList.add("highlight")
element.classList.remove("hidden")
element.classList.toggle("active")
